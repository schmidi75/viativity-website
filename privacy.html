2. Webseiten-Hosting (GitHub Pages)

Diese Webseite wird auf GitHub Pages gehostet (GitHub Inc., USA). Beim Besuch der Seite erhebt GitHub automatisch Informationen, die dein Browser übermittelt (Server-Log-Files). Dazu gehören:

  • IP-Adresse deines Geräts
  • Datum und Uhrzeit des Zugriffs
  • Name und URL der abgerufenen Datei
  • Browsertyp und Betriebssystem

Diese Daten dienen der Sicherheit und Stabilität der Seite. Ich nutze keine externen Tracking-Tools wie Google Analytics und setze keine Marketing-Cookies ein.

3. Viativity™ App

Die App Viativity™ respektiert deine Privatsphäre konsequent:

  • Sensordaten: Alle für die Funktionen (z.B. künstlicher Horizont) benötigten Sensordaten deines Smartphones werden ausschließlich in Echtzeit lokal auf deinem Gerät verarbeitet.
  • Keine Speicherung: Es werden keine Bewegungs- oder Standortprofile erstellt oder gespeichert.
  • Kein Cloud-Senden: Es findet kein Datentransfer deiner persönlichen Nutzungsdaten an externe Server statt.
